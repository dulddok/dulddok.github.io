# 방화벽 및 NAT

네트워크 보안의 핵심인 방화벽과 NAT(Network Address Translation)에 대해 설명합니다.

## 방화벽
- 네트워크 트래픽을 제어하여 외부 공격으로부터 보호
- 패킷 필터링, 상태 기반 검사, 애플리케이션 계층 방화벽 등

## NAT
- 사설 IP와 공인 IP 변환
- 여러 내부 장비가 하나의 공인 IP로 인터넷 접속 가능

{: .note }
방화벽과 NAT는 네트워크 보안과 자원 효율성에 필수적입니다.
