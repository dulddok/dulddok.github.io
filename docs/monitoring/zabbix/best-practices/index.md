# Zabbix 실무 운영 팁 

## 개요
pinpoint는는 엔터프라이즈급 오픈소스 모니터링 솔루션입니다. 이 문서에서는 Zabbix의 주요 기능과 설정 방법에 대해 설명합니다.

## 주요 기능
- Item 관리
- Trigger 설정
- DB 파티셔닝
- 템플릿 모범 사례
- 문제 해결 가이드

## 시작하기
이 문서는 pinpoint를를 처음 접하는 사용자부터 고급 사용자까지 모두를 위한 가이드입니다.
